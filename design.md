# Portfolio Design Style Guide

## Design Philosophy

### Color Palette
- **Primary Colors**: Deep navy blue (#1a202c), rich purple (#553c9a), vibrant violet (#805ad5)
- **Accent Colors**: Soft lavender (#e9d8fd), pure white (#ffffff)
- **Background**: Consistent dark gradient from deep navy to rich purple
- **Text Colors**: White (#ffffff) for primary text, light gray (#e2e8f0) for secondary text

### Typography
- **Display Font**: "Playfair Display" - elegant serif for headings and logo
- **Body Font**: "Inter" - clean, modern sans-serif for readability
- **Font Hierarchy**: 
  - H1: 3.5rem (56px) - Hero headlines
  - H2: 2.5rem (40px) - Section headers
  - H3: 1.875rem (30px) - Project titles
  - Body: 1rem (16px) - Regular text
  - Small: 0.875rem (14px) - Captions and metadata

### Visual Language
- **Aesthetic**: Modern professional with sophisticated dark theme
- **Mood**: Confident, innovative, trustworthy
- **Style**: Clean minimalism with strategic use of gradients and subtle animations
- **Imagery**: High-quality professional product interfaces and abstract tech visuals

## Visual Effects & Styling

### Used Libraries
- **Anime.js**: Smooth page transitions and element animations
- **Typed.js**: Dynamic typewriter effect for hero text
- **Splitting.js**: Advanced text animation effects
- **ECharts.js**: Data visualization for project metrics
- **Splide.js**: Image carousels and sliders
- **p5.js**: Creative background effects and interactive elements

### Animation Effects
- **Hero Section**: Typewriter animation with gradient color cycling
- **Project Cards**: 3D tilt hover effects with image zoom
- **Navigation**: Smooth slide transitions with active state indicators
- **Scroll Animations**: Fade-in reveals with staggered timing
- **Background**: Subtle particle system with floating geometric shapes

### Header Effects
- **Navigation Bar**: Glass morphism effect with backdrop blur
- **Logo**: Subtle glow animation on hover
- **LinkedIn Icon**: Smooth color transition and scale effect
- **Menu Items**: Underline animation on hover

### Interactive Elements
- **Buttons**: Gradient background with hover lift effect
- **Form Fields**: Floating label animations with focus states
- **Project Images**: Ken Burns effect with overlay text reveal
- **Cards**: Depth shadow expansion on hover

### Layout & Grid System
- **Grid**: CSS Grid with responsive breakpoints
- **Spacing**: Consistent 8px base unit system
- **Containers**: Max-width 1200px with centered alignment
- **Responsive**: Mobile-first approach with fluid typography

### Background Design
- **Primary Background**: Linear gradient from #1a202c to #553c9a
- **Decorative Elements**: Subtle geometric patterns and floating particles
- **Section Dividers**: Gradient lines and shape overlays
- **Depth**: Layered elements with varying opacity and blur effects

### Hover Effects
- **Project Cards**: 
  - Image scale: 1.05x
  - Overlay gradient fade-in
  - Text color transition to accent purple
  - Box shadow expansion
- **Navigation Links**:
  - Underline width animation
  - Color transition to violet
  - Smooth 0.3s ease transition
- **Buttons**:
  - Background gradient shift
  - Transform translateY(-2px)
  - Box shadow intensity increase

### Mobile Responsiveness
- **Breakpoints**: 
  - Mobile: 320px - 768px
  - Tablet: 768px - 1024px
  - Desktop: 1024px+
- **Navigation**: Hamburger menu with slide-out drawer
- **Grid Layouts**: Single column on mobile, multi-column on larger screens
- **Typography**: Fluid scaling based on viewport size
